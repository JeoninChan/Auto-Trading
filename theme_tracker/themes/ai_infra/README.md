# AI 인프라 테마

## 개요

- 하이퍼스케일러(Microsoft, Amazon, Google) 데이터센터 CapEx 발표
- 전력망 병목 관련 규제/법안
- 냉각 기술 계약 (Vertiv, Modine 등)
- NVIDIA GPU 수급 이슈 → 공급망 수혜주
